##Movie Recommender System  

A simple movie recommendation app built with Python and Streamlit.  
Users can select their preferred movie genres and receive suggested movies based on similarity.

##Description
This project recommends movies based on user inputs such as selected genres.  
The dataset includes movie titles, genre labels, and description info.  
The recommender uses a simple content-based recommendation model.

---

##Features
Dataset loading  
Genre based filtering  
User-friendly Streamlit interface  
Multiple genre selection  
Test dataset included
